# Version 1.0

La version 1.0 marque le premier contrat stable de Tapo Event Bridge.

## Garanties de stabilité

- Le schéma 1 du payload de tableau de bord reste rétrocompatible pendant la branche 1.x.
- Les événements sont conservés uniquement dans un tampon mémoire borné.
- Les transitions d'état dupliquées peuvent être supprimées sans interroger les caméras.
- La découverte lit uniquement les registres Home Assistant et ne réveille pas les caméras sur batterie.

## Nouveaux diagnostics

Le capteur **Métriques d’exécution** expose la durée de fonctionnement, la taille du
tampon, les doublons supprimés, les écouteurs, les nettoyages et les transports actifs.
Le bouton **Effacer l’historique des événements** vide uniquement le tampon mémoire de
l’intégration ; il ne supprime pas l’historique Recorder de Home Assistant.
