# Explorateur de capacités

Le capteur `Explorateur de capacités` expose un profil respectueux de la vie privée pour chaque caméra Tapo découverte.

Il utilise uniquement les registres des appareils et entités de Home Assistant. Il n’effectue aucun polling réseau et ne réveille pas les caméras sur batterie.

Chaque profil contient les informations observées sur le modèle et le firmware, les compteurs d’entités, les plateformes sources, les capacités déduites, leur niveau de preuve, une indication d’alimentation et un score de complétude. Les données absentes restent explicitement inconnues.
