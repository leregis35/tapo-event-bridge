# Analyse du parc

L’analyse du parc regroupe les informations déjà visibles dans les registres Home Assistant pour toutes les caméras découvertes. Elle expose les répartitions par modèle, alimentation, plateforme, domaine d’entités et couverture des capacités, ainsi qu’une liste prudente des caméras qui manquent de preuves.

Aucune interrogation réseau n’est effectuée et aucune caméra sur batterie n’est réveillée. Une information absente reste inconnue.
