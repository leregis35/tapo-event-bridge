# Centre de supervision du parc

Le capteur Centre de supervision du parc fournit une vue compacte de toutes les caméras. Il combine les informations du registre Home Assistant avec les événements normalisés déjà observés par Tapo Event Bridge.

Ses attributs contiennent notamment la santé du parc, les détections actives, la répartition des événements, la caméra la plus active, le dernier événement et un résumé borné pour chaque caméra.

La liste est volontairement limitée à 12 caméras pour rester compatible avec Home Assistant Recorder. Les preuves détaillées restent disponibles dans les diagnostics de l’intégration.
